## 🌐 Deployment

This website is hosted using GitHub Pages.

To deploy:

1. Push the project to GitHub.
2. Open Repository Settings.
3. Go to Pages.
4. Select:

   * Branch: main
   * Folder: /(root)
5. Click Save.
6. Wait a few minutes for the website to go live.
