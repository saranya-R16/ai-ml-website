# Data Analysis & AI/ML Website

A modern and responsive website built using HTML and CSS that showcases the concepts of Data Analysis and Artificial Intelligence/Machine Learning. The website includes an attractive user interface, technology-themed design, image gallery, statistics section, and smooth navigation.

## 🚀 Live Website

https://saranya-r16.github.io/ai-ml-website/


## 📌 Features

* Responsive Design
* Modern Navigation Bar
* Attractive Hero Section
* Data Analysis Information Section
* Artificial Intelligence & Machine Learning Section
* Statistics Dashboard
* Technology Image Gallery
* Contact Information Section
* Stylish Footer
* Mobile-Friendly Layout

## 🛠 Technologies Used

* HTML5
* CSS3
* Google Fonts (Poppins)
* Font Awesome Icons
* Git & GitHub
* GitHub Pages
